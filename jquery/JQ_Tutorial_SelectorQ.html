LC0841 Teamwork and Communication Skills

Aims to equip students with the necessary teamwork and communication skills relevant to the efficient and successful completion of digital media and infocomm technology projects and provide them with the foundational skills for effective written and oral communication. Through integrated tasks, students will practise and demonstrate the skills of team-building and formation, participation in and management of teams, appropriate communication strategy design and effective written and oral communication.

ST1001 Database Development

Aims to equip students with the database knowledge including the characteristics of a relational model, functions of relational database management systems (RDBMS), process of normalization, entity-relationship modeling, database system development cycle, database security functions as well as practical skills in Structured Query Language (SQL).

ST1008 Web Client Development

Provide students with the knowledge and skills in developing effective web pages using Hypertext Markup Language (HTML) and Cascading Style Sheets (CSS). They will learn how to code web pages according to standards set by the World Wide Web Consortium (W3C). They will also learn how to use JQuery in order to add greater functionality to their websites
